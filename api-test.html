医生API接口测试

API接口说明

以下API接口已实现，用于替代原来的JSON文件数据源：

GET
http://localhost:8080/api/doctors/active

获取所有活跃医生列表（与原来JSON格式相同）

GET
http://localhost:8080/api/doctors

获取所有医生列表（包括非活跃）

GET
http://localhost:8080/api/doctors/statistics

获取医生统计信息

GET
http://localhost:8080/api/doctors/{id}

根据ID获取医生详情

GET
http://localhost:8080/api/doctors/username/{username}

根据用户名获取医生详情

前端更新说明

前端需要修改以下位置，从原来的JSON文件改为调用API：

  1. 修改 src/store/index.ts 中的 getActiveDoctors() 方法
  2. 修改 src/store/index.ts 中的 getStatistics() 方法
  3. 修改 src/store/index.ts 中的 getDoctorByUsername() 方法
  4. 更新所有相关组件的导入方式

数据库配置

已创建完整的MySQL数据库配置：

  • 数据库文件: docker-compose.yml - 包含MySQL和phpMyAdmin配置
