### 2. 功能开发

#### 2.1 语言切换功能
```
修复语言切换双向不生效的问题。当前问题：完成中文→英文切换后，从英文切换回中文不生效。

问题分析：LanguageSwitcher.vue中的currentLocale通过computed(() => getLocale())计算，但getLocale()从localStorage读取，导致状态不一致。

请提供解决方案，统一使用useLocale钩子获取响应式语言状态。
```

## 开发经验总结

### 1. 有效的提示词特征
1. **明确的目标**：清晰说明要实现什么功能
2. **具体的技术要求**：指定技术栈、框架、版本
3. **完整的上下文**：提供项目背景和现有代码状态
4. **验收标准**：明确如何验证功能是否完成
5. **示例和模板**：提供参考代码或文档模板

### 2. 避免的问题
1. **模糊的需求描述**：导致AI理解偏差
2. **技术栈不明确**：导致不兼容的代码生成
3. **缺少上下文**：AI无法理解项目现状
4. **不完整的验收标准**：难以验证功能是否完成

### 3. 最佳实践
1. **分步骤提示**：复杂功能拆分成多个步骤
2. **先设计后实现**：先要求设计文档，再实现代码
3. **包含测试要求**：要求同时提供测试代码
4. **要求文档**：功能完成后要求更新相关文档
